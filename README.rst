# face_detector

This opencv project helps to detect face from an imported image, live image using webcam, recorded video, live stream video using pretrained models in opencv, dlib and python 

Installation
----------------

For better experience, perform the installation in a virtual environment

Install Pillow

.. code-block:: text

    $ pip install pillow

Install Imutils

.. code-block:: text

    $ pip install imutils

Install Cmake

.. code-block:: text

    $ pip install cmake

Install dlib

.. code-block:: text

    $ pip install dlib

Install Face_Recognition

.. code-block:: text

    $ pip install face_recognition

Install Opencv

.. code-block:: text

    $ pip install opencv-python
